Es el código que se utiliza para crear la estructura de una página web y su contenido, hace uso de etiquetas y atributos que van formando el sitio web y organizan la forma en que los usuarios podrán visualizarla.

Estrutura HTML

<!DOCTYPE html>
Indica el tipo de lenguaje que estamos utilizando, en el caso de html5 esta es la forma para indicarlo.

<head>
Es el que contiene el titulo de nuestra página web, aunque su contenido no es visible para el navegador. Se encuentra en una posición antes que la de <body>.

<title>
Es es título del documento y se mostrará solamente en la pestaña del navegador, no como contenido.

<meta charset="utf-8">
Indica la codificación o juego de caracteres a utlizar, en el caso html5 utilizaremos siempre UTF-8.

<body>
Es muy importante en la estructura html ya que en el incluiremos todo el contenido de nuestra página, todas ellas incluidas entre las etiquetas de apertura y cierre de <html>

Es importante destacar que HTML será utilizado para indicar si el elemento se trata de un párrafo, un título, si es un texto a resaltar y debe ir en negrita o cursiva.

Para las cuestiones de aspecto, es decir fuente con la que se mostrará el sitio web, color y más cuestiones utilizaremos CSS.

CSS es el lenguaje utilizado para dotar de cualidades visuales y estéticas a una página web. Abarca cuestiones relativas a fuentes, colores, márgenes, líneas,altura, anchura, imágenes de fondo, posicionamiento avanzado y muchos otros más temas que iremos viendo a lo largo del curso.

CSS sirve también para mejorar la accesibilidad de las páginas ya que al separar el diseño del contenido facilita a los navegadores la interpretación del código y también ayuda con el SEOo posicionamiento del sitio ya que permite que los motores de búsqueda rastreen el contenido de una página web con mayor precisión, lo que mejora su posicionamiento en los resultados de búsqueda.

Estructura CSS

Selector
El selector es el elemento HTML al que vamos a aplicarle un estilo concreto.

El bloque de declaración
Contiene una o más delcaciones separadas por un punto y coma. Cada declación incluye un nombre de propiedad CSS y un valor, separados por dos puntos.

Propiedad
La propiedad es cada una de las diferentes características que brinda el lenguaje CSS.

Valor
Cada propiedad tiene una serie de valores concretos que puede adoptar, con los que tendrán uno u otro comportamiento.
